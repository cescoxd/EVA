Etiquetas HTML

Body y sus atributos

Este elemento es parte de la estructura básica de un documento y donde se pone todo el contenido de la página.

Este elemento tenía varios atributos en versiones antiguas de HTML. Estos atributos se han dejado de utilizar y por lo tanto se han quitado, ahora tienen atributos generales.

P y sus atributos

Esta etiqueta se utiliza para escribir parágrafos en las páginas web.

Es importante acordarse de cerrar las etiquetas ya que sinó se quedan abiertas y no funcionan como deberían.

Esta etiqueta no tiene etiquetas específicas, solo tiene atributos generales: id (Para darle una identidad a la etiqueta) ,style (Para darle estilo) ,class (Para asignar una clase) ,dir (Es para indicar cómo se va a leer la lengua, "ltr" de izquierda a derecha o "rtl" de derecha a izquierda) ,lang (Código del idioma)

a y sus atributos

Este elemento crea enlaces a diferentes páginas o secciones de la misma página. Los vínculos son la parte más esencial de HTML y una de las razones principales por las que ha nacido. Los vínculos pueden ser utilizados, para enlazar otros documentos, y también para hacer referencia a una parte específica de su contenido.

Atributos principales: href: La url de otra página otra parte de la página etc... ,target: Le indicas donde quieres que se se dirija el enlace que le has indicado con el atributo href. Este atributo también sirve para abrir el enlace a una pestaña nueva, se pondría como se ve en la captura de pantalla de la izquierda.

img y sus atributos

Este elemento representa a una imagen, lo cual es un recurso externo que se inserta en el cuerpo de un documento. Es unas de las pocas etiquetas que no hace falta cerrar.

Atributos principales: src: Indica la URL de la imagen a mostrar. ,alt: Establece un texto alternativo que describa la imagen.Atributo obligatorio).

div y sus atributos

Este elemento es una etiqueta sin un significado semántico.

Se utiliza para dividir secciones y tener organizado el código.

Los autores deberían considerar a esta etiqueta como último recurso, reservado únicamente para aquellos casos en los que elementos con mayor significado, como main, header, footer, nav o article no son aplicables.
